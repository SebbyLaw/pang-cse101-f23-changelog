Objectives:

Implement the List ADT and a write test code.

Description:

In this assignment you will implement a List ADT and write test code to verify that your List ADT works as expected.

Details:

  • You are provided with stubs for the List ADT in List.h. This file contains functions that the List ADT need to support. Do not edit this file.

  • The function headers may look overly complicated but that's because it's designed to support Lists of different data types. Note that (void *) can be cast to the appropriate data type needed by your application. Also, functions are being passed as parameters. This is likely a new concept for most of you. Here's a link that explains this concept with some examples.

  • There are 2 main task for this homework:

    • Implement the functions specified in List.h. You will write your code for that in List.c.

    • Test your List ADT.
      • An application code is provided to help you verify your implementation of the functions of this List ADT. There are a couple of questions in the code that you should answer for yourself. In addition, you should add additional test for the deleteList function.
      • The expected output for the starter test code hwk1.c is in hwk1.out.

  • Files for this program: The following files are provided to you, and should all be submitted with your work.

    • Makefile: Makefile for compiling your code.

    • List.h: Do not change this file. It has a list of functions that you need to implement for the List ADT

    • List.c: This is where you implement your List ADT.

    • hwk1.c: This is where you put your client app.
