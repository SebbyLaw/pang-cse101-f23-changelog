Music Database Manager Assignment

Due before midnight, Sunday, November 19, 2023
Late submissions will not be accepted/graded

Objective:

To create a program that allows users to manage a music database with functionalities such as adding, removing, updating, finding, and displaying songs, using the Dictionary ADT.

Description:

You are tasked with developing a simple music database management application using your Dictionary.c, HashTable.c and the provided header files Dictionary.h, HashTable.h.

At the start, your application will read input from stdin. The stdin input will have a filename to read the songs and their authors, as well as different commands to execute post initialization. After reading the input and the initialization phase, your application should provide the following functionalities according to the index of commands:

  1. Add a Song: This allows users to input a song name and its corresponding author, and add them to the dictionary, which is being used to add songs to the database.
    Example: Input '1', then 'Shape of You', and then 'Ed Sheeran' to add the song to the database.
  2. Remove a Song: This enables users to delete a song from the database using the song's name.
    Example: Input '2' and then 'Shape of You' to remove the song from the database.
  3. Update Song Details: This lets users update the author of a song using the song's name.
    Example: Input '3', then 'Photograph', and then 'Unknown' to update the author's name.
  4. Find the author of a song: This allows users to find a song in the database using the song's name.
    Example: Input '4' and then 'Photograph' to find the author of the song.
  5. Display All Songs: This will print out all the songs and their authors in the database to stdout.
    Example: Input '5' to display all songs and their authors.
  6. Exit: This will terminate the program.
    Example: Input '6' to exit the program.
  7. Top Composer: Go over the entire song database to find the top composer who has the most songs. In case of ties, print out all composers.
    Example: Input '7' to display the top composer or composers in case of a tie.
  8. Most Popular Word: Go over every song name and figure out the most popular word. In case of ties, print out all the songs. Printing out is gonna be on stdout.
    Example: Input '8' to find the most popular word in song names.

Requirements:

  1. Implement the ht_hash function in HashTable.c
  2. Use the Dictionary data structure to store songs and their authors.
  3. Ensure that all functions in Dictionary.c are tested and utilized.
  4. Write a pg3 file where you will implement your music database using the dictionary.
  5. Handle potential errors gracefully such as trying to remove a song that doesn't exist.
  6. Ensure that the program continues running until the user chooses to exit.
