Objective:

To create a program that allows users to manage a music database with functionalities such as adding, removing, updating, finding, and displaying songs, using the Dictionary ADT.

Description:

You are tasked with developing a simple music database management application using your Dictionary.c, HashTable.c and the provided header files Dictionary.h, HashTable.h.

At the start, your application will read input from stdin. The stdin input will have a filename to read the songs and their authors, as well as different commands to execute post initialization. After reading the input and the initialization phase, your application should provide the following functionalities according to the index of commands:

  1. Add a Song: This allows users to input a song name and its corresponding author, and add them to the dictionary, which is being used to add songs to the database.

    Example: Input '1', then 'Shape of You', and then 'Ed Sheeran' to add the song to the database.

  2. Remove a Song: This enables users to delete a song from the database using the song's name.

    Example: Input '2' and then 'Shape of You' to remove the song from the database.

  3. Update Song Details: This lets users update the author of a song using the song's name.

    Example: Input '3', then 'Photograph', and then 'Unknown' to update the author's name.

  4. Find the author of a song: This allows users to find a song in the database using the song's name.

    Example: Input '4' and then 'Photograph' to find the author of the song. If the song is found, the output should be in the format 'Found song: SONG_NAME by AUTHOR_NAME'. If the song is not found, the output should be 'Song not found!'.

    Output Example if song is found: 
    Found song: Photograph by Unknown
    Output Example if song is not found: 
    Song not found!
  5. Display All Songs: This will print out all the songs and their authors in the database to stdout.

    Example: Input '5' to display all songs and their authors. The format for each song should be 'SONG_NAME by AUTHOR_NAME' on a new line for each song.

    Output Example: 
    i believe of by frankie laine
mohabbat bhi jhoothi by Ed Sheeran
Photograph by Unknown
patricia by frankie laine
cry by Ed Sheeran
  6. Top Composer: Go over the entire song database to find the top composer who has the most songs. In case of ties, print out all composers.

    Example: Input '6' to display the top composer or composers in case of a tie. The format should list each author followed by the count of songs in the format 'AUTHOR_NAME: COUNT song(s)'.

    Output Example: 
    Ed Sheeran: 2 song(s)
frankie laine: 2 song(s)
  7. Most Popular Word: Go over every song name and figure out the most popular word. In case of ties, print out all the songs. Printing out is gonna be on stdout.

    Example: Input '7' to find the most popular word in song names. The format should list each word followed by the count of occurrences in the format 'WORD: COUNT occurrence(s)'.

    Output Example: 
    jhoothi: 1 occurrence(s)
believe: 1 occurrence(s)
mohabbat: 1 occurrence(s)
bhi: 1 occurrence(s)
of: 1 occurrence(s)
i: 1 occurrence(s)
patricia: 1 occurrence(s)
cry: 1 occurrence(s)
  8. Exit: This will terminate the program.

    Example: Input '8' to exit the program. The output should be a simple confirmation message.

    Output Example: 
    Goodbye!
  9. Invalid Input Handling: If the input number is not between 1 and 8, prompt the user to make a valid selection.

    Output Example for invalid input: 
    Invalid choice! Please select a valid option.

Requirements:

  1. Implement the ht_hash function in HashTable.c
  2. Use the Dictionary data structure to store songs and their authors.
  3. Ensure that all functions in Dictionary.c are tested and utilized.
  4. Write a prog3 file where you will implement your music database using the dictionary.
  5. Handle potential errors gracefully such as trying to remove a song that doesn't exist.
  6. Ensure that the program continues running until the user chooses to exit.
